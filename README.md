hue-queue is a high-speed digital workflow designed for live event artists (watercolorists, caricaturists, etc.) who need to manage high-volume guest portraits (25+ per hour) without relying on unreliable venue Wi-Fi.

Key Technical Features:
* Local-First Architecture: Runs on a local 5GHz subnet for zero-latency image delivery.
* Automated Ingress: Monitors a "Hot Folder" from tethering software (Lightroom/Sony/Canon) for instant processing.
* Dynamic Watermarking: Automatically sequences and stamps guest numbers on photos for physical-to-digital cross-referencing.
* State Management: Minimalist web-based interface for artists to "Claim" and "Mark Done" without a complex database.

GPLv3 Licensed: Built to stay open and accessible.
